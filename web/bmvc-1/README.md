# BMVC I - Carlisle Automóveis

Projeto de página estática com HTML, CSS e JavaScript puro, seguindo a arquitetura BMVC.

## Requisitos

Apenas um navegador web moderno.

## Como Usar

1. Abra o arquivo `index.html` em seu navegador
2. Interaja com os botões e formulário
3. Teste a responsividade redimensionando a janela

## Funcionalidades

- Menu de navegação
- Seção hero com chamada para ação
- Listagem de carros
- Formulário de contato com validação
- Design responsivo
- Navegação suave entre seções

## Arquitetura BMVC

**View (index.html):** Estrutura HTML semântica
**Styling (style.css):** Estilos responsivos com Flexbox e Grid
**Controller (script.js):** Lógica de interatividade

## Validações

- Email deve estar em formato válido
- Todos os campos do formulário são obrigatórios
- Confirmação antes de ações importantes

## Responsividade

Funciona perfeitamente em:
- Desktop (> 768px)
- Tablet (480px - 768px)
- Mobile (< 480px)
