# BMVC II - Carlisle Automóveis

Projeto CRUD completo com Backend em Python (Flask) e Frontend em HTML/CSS/JS.

## Requisitos

- Python 3.7+
- Flask

## Instalação

```bash
pip install flask
```

## Como Executar

```bash
python app.py
```

Abra http://localhost:5000 no navegador.

## Funcionalidades

- Listar carros (READ)
- Adicionar novo carro (CREATE)
- Editar carro (UPDATE)
- Deletar carro (DELETE)
- Validação de campos
- Design responsivo

## Arquitetura

- **Backend:** Flask (app.py)
- **Model:** Classe Carro (models.py)
- **Frontend:** Templates Jinja2
- **Estilos:** CSS responsivo
- **Interatividade:** JavaScript puro
